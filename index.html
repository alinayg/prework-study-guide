HTML

    Head elements

    • meta = contains metadata for browsers.
    • title = what is shown in the browser top bar to the user.

    Body elements

    • The body element represents the visable content shown to the user.
    • There are various elements that can be used to format the body of the HTML.
    • h1-h6 = used to format the body into sections with larger text starting at H1 and progressivly getting smaller at H6.
    • Main = contains the main body of the HTML document.
    • Header = contains the webpage information and meta data, along with other information for the web browsers.
    • section = allow us to elegantly organize the content on our page based on the topics we have. attributes, id, class. HTML uses the id attribute as a way to identify elements and select them, used with CSS.
    • Within tags there are other veriables that you can set, some are requiored, some are not.
    • Footer is where most web developers put thair contact information or signatures.

    Main elements

    • ul = unorderd list, will make a list with dots.
    • ol = orderd list, will make a list with numbers.
    • li = is used to add items to lists.
    • img = adds images to the body, requires the scr attribute that points to the image source. There are other optional attributes.
    • br = this element creates a line of empty space, or a line break, between two blocks of content.
    • a = anchor = hyperlink = creates links to the same webpage or other webpages. The href attribute points to the URL for the link.
    • p = represents a paragraph or block of text.

    Footer elements

    • Same as main.
